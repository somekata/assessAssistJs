# 学会審査アシスタント

ローカル (オフライン) で学会演題を評価・集計できる、シンプルなブラウザアプリです。  
サーバーやクラウドにデータを送信しない設計なので、各審査員のPCだけで安全に採点を進められます。

---

## 特徴

- **data.csv (演題リスト)** を読み込むと、その内容が画面に一覧として表示されます  
  - 各演題には ID / タイトル / 抄録 が含まれます
- 審査員は各演題に対して
  - スコア（1〜5）
  - コメント
  - 状態（未保存 / 保存済み / 確定）
  を入力できます
- 進捗状況は色で表示されます  
  - 赤：まだ「保存」していない  
  - 黄：保存した（スコア・コメントは入った）が未確定  
  - 青：確定済み（ロック済み）
- 審査が終わったら、自分の採点結果だけを CSV でダウンロードして事務局に提出できます
- 評価データはブラウザの `localStorage` のみに保存され、外部に自動送信されません

---

## セキュリティと運用上の注意 (重要)

**演題そのものの情報 (data.csv) は機密扱いです。GitHubなど公開環境に置かないでください。**

- このリポジトリには実データは含めません  
  → かわりに `sample.json` を参考例として同梱しています
- data.csv は学会事務局などから、審査員にのみ直接配布してください（メール添付・USBなど）
- 審査員は配布された data.csv をローカルで読み込んで使います。外部にアップロードする必要はありません
- 採点結果は審査員の PC のブラウザ内 (localStorage) に保存されます  
  → ネットワーク送信はされません
- 共有PCでは使わないでください。個人PCでの利用を推奨します
- 審査が終わったら「全データ初期化」ボタンで localStorage 内の採点データを削除してください
